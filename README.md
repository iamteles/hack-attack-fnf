You can use Tr1Ngle Engine as long as you credit me.

Features:
		
	Misses info;
	
	Accuracy info;
		
	Customizable keybinds;

	Modifiers;

	Note splashes;

	Change Difficulty in pause menu;

	Downscroll;

	Shaders;

	Middlescroll;

	Blue Balled Counter in pause menu;

	Time Left Counter;

	Memory Counter;
	
	Colors for Health Bar;

	Freeplay BG color changing for each song;

	And More;
